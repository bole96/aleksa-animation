# C# zadatak – vežba

Ovo je jednostavan C# projekat rađen u okviru vežbi, sa ciljem učenja osnova programiranja i rada u Visual Studio okruženju.

## Tehnologije
- C#
- .NET
- Visual Studio

## Opis
Projekat je namenjen vežbanju:
- osnovne C# sintakse
- rada sa klasama i objektima
- osnovne logike programa
- razumevanja toka izvršavanja programa

Zadatak je rađen kao deo učenja i može se dodatno proširivati i unapređivati.

## Kako pokrenuti
1. Preuzeti ili klonirati repozitorijum
2. Otvoriti projekat u **Visual Studio**
3. Pokrenuti aplikaciju (Start / F5)

## Status
Projekat je edukativnog karaktera i služi za vežbu i usavršavanje znanja iz C# jezika.
